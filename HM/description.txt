Все дополнительные модули написанные с нуля вынесены в модуль Extra.
В рамках диплома были также произведены изменения в коде эталонного декодера.
Для декодера были добавлены дополнительные ключи:
-а - алгоритм компенсации искожений:
    1 - без компенсации(по умолчанию)
    2 - копирование с предыдущего кадра
    3 - интерполяция Лагранжа
    4 - склеивание кадров для видео с бпла(дрона)
Последний алгоритм также требует дополнительного параметра
-m - файл с координатами кадров

NALinfo [input.bin] [res.txt] - записывает в файл res.txt информацию из сжатого потока
PackageLost [input.txt] [output.txt] [prob] - удаляет из входного текстового файла prob строк
LoseBits [input.bin] [nalInfo.txt] [output.bin] - оставляет во входном потоке описанные в текстовом файле пакеты
